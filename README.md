# Acak Kelompok (Flutter)

Aplikasi Flutter sederhana untuk mengacak kelompok dari daftar nama. Mendukung input banyak nama, penentuan jumlah kelompok, copy hasil, serta export ke PDF dan Excel. Cocok untuk tugas kuliah dan presentasi.

## Run
flutter pub get && flutter run

## Tech
Flutter · Dart · excel · pdf · printing · share_plus
